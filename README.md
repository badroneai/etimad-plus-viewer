# كشّاف — مرآة مستودع اعتماد بلس

واجهة عربية ثابتة تعرض **كل ما جُلب** من مستودع اعتماد بلس (مجمّع غير رسمي)، لا اختصار للطبقات القابلة للعرض.

## المعاينة

https://badroneai.github.io/etimad-plus-viewer/

## ماذا يُعرض

| المجموعة | المصدر | العدد |
|---------|--------|------:|
| منافسات مفتوحة | `81_tenders_open` | 1,604 |
| خلال 7 / 30 يوماً | layers 81 | 782 / 1,580 |
| مرساة | `81_tenders_awarded_yes` | 11,300 (جزئي) |
| عيّنة SSR منافسات | layer 62 | 50 |
| أنشطة / جهات / أنواع facets | 83 / 86 / 87 | 138 / 802 / 9 |
| جهات API / SSR / خريطة | 60 / 64 / 09 | 600 / 600 / 696 |
| شركات API / SSR / خريطة | 61 / 63 / 08 | 60 / 200 / 20,968 |
| مراجع منافسات الخريطة | layer 10 | 13,665 |
| أنشطة الخريطة | layer 07 | 82 |
| تصنيف مرصود + حالة الجلب + تدقيق | 72 / meta | ملفات كاملة |

## التفاصيل

الضغط على منافسة يفتح بطاقة محلية بكل الحقول المجلوبة. زر اعتماد الرسمية منفصل للمشاركة فقط.

## تحديث البيانات من المستودع

```bash
python scripts/export_warehouse.py
```

## تشغيل محلي

```bash
python -m http.server 8080
```

افتح `http://localhost:8080`.
